sodorg_renewal.utils.wrap_molecule

sodorg_renewal.utils.wrap_molecule#

sodorg_renewal.utils.wrap_molecule(atoms: ase.atoms.Atoms, max_dist: float = 0.5)[source]#

Translates molecule such that no bit of the molecule sticks out passed max_dist in any direction.

max_dist is in fractional units (default is 0.5).

returns the atoms object with the molecule wrapped
